Frequência

Nesta rotina são cadastradas todas as frequências utlizadas no sistema.
Todo programa de autocontrole necessita ser respondido em períodos de tempo ( frequência )
Todo programa cadastrado no sistema deve ser associado a uma frequência, caso contrário, o Cadastro de Programas não será concluído.

O cadastro de frequência é composto pelas seguintes informações:

  • ID - Código identificador da frequência, este campo é preenchido automaticamente pelo sistema.
  • Descrição - Descrição da frêquencia cadastrada, por exemplo: DIARIA, MENSAL etc..
