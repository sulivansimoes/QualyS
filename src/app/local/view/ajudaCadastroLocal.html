Local

Nesta rotina são cadastrados todos os locais utlizados no sistema.
Todo formulário de perguntas necessita ser associado à um local que corresponde ao lugar que o fomulário em questão faz referência.
Obrigatóriamente todo fomulário cadastrado no sistema deve ser associado a um local, caso contrário, o Cadastro de Fomulários não será concluído.

O cadastro de Local é composto pelas seguintes informações:

  • ID - Código identificador do Local, este campo é preenchido automaticamente pelo sistema.
  • Descrição - Descrição do Local cadastrado, por exemplo: DEPOSITO DE MATERIA PRIMA, BANHEIRO, TACHO etc..
