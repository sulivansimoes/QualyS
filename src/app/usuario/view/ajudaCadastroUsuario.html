Usuário

Nesta rotina são cadastrados todos os usuário que tem acesso ao sistema.

O cadastro de Usuário é composto pelas seguintes informações:

  • CPF - Número do cpf do usuário (Preencha somente com os números do CPF, sem os pontos)
  • Nome - Nome do usuário do sistema.
  • Senha - Senha que usuário fará login no sistema.
  • E-mail - Informe o e-mail do usuário.
  • Bloqueado - Informe se o usuário se encontra bloqueado ( Ativo ) ou não no sistema.
  • Assinatura - Informe uma imagem contendo a assinatura do usuário.
